Install Headroom

Version: v0.1.0 · Requires: Chrome 111+ (or any Chromium browser — Edge, Brave, Arc)

Headroom isn't on the Chrome Web Store yet, so for now you install it by loading it unpacked — Chrome's built-in way to run an extension straight from a folder. It takes about a minute.

Download headroom-v0.1.0.zip ↓ ~15 KBSource-available · audit it on GitHub

Step by step

  1. Download the zip

    Grab headroom-v0.1.0.zip using the button above.

  2. Unzip it

    Chrome can't load a .zip directly — you have to extract it first. On Windows, right-click the file → Extract All. On macOS, double-click it. You'll get a folder named headroom-v0.1.0 containing manifest.json.

  3. Open the extensions page

    Type chrome://extensions into your address bar and press Enter.

  4. Turn on Developer mode

    Flip the Developer mode toggle in the top-right corner of that page.

  5. Load unpacked

    Click Load unpacked, then select the unzipped headroom-v0.1.0 folder — the one that contains manifest.json. Headroom now appears in your extensions list.

  6. Try it

    Open claude.ai and send a message. The headroom badge appears in the bottom-right, showing your 5-hour and 7-day usage.

Seeing a "Developer mode extensions" warning? That's normal for any unpacked extension and isn't an error — it goes away once Headroom ships to the Chrome Web Store. The code is source-available, so you can read exactly what it does first.

Updating later: download the newer zip, unzip it, and on chrome://extensions either click the refresh ↻ icon on the Headroom card or remove it and load the new folder.

Is it safe?

Headroom has no backend, makes no network requests of its own, and only ever reads the quota event claude.ai already sends to your browser. Nothing leaves your device. The full details are in the privacy policy, and every line is auditable on GitHub.
